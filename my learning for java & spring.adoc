* ``cmd``에서 실행한다. ``PowerShell``에서 사용시 예상치 못한 결과들이 종종 나온다.
----

    -DgroupId=kr.cloudscape.java ^
    -DartifactId=java-programming ^
    -DarchetypeArtifactId=maven-archetype-quickstart ^
    -DinteractiveMode=false
----
